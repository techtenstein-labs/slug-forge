# slug-forge API

Free slugify API with Unicode transliteration (accents, umlauts, ß, æ, œ, ø), configurable separator, strict mode, and length cap. Batch endpoint for many strings at once.

- Live: https://slug-forge.techtenstein.com
- OpenAPI: https://slug-forge.techtenstein.com/openapi.json
- Health: https://slug-forge.techtenstein.com/health

## Endpoints

See [openapi.yaml](openapi.yaml).

## Example

```bash
curl https://slug-forge.techtenstein.com/health
```
